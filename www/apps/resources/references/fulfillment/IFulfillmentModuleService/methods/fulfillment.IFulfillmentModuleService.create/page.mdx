# create - Fulfillment Module Reference

## create(data, sharedContext?): Promise&#60;[FulfillmentSetDTO](../../../interfaces/fulfillment.FulfillmentSetDTO/page.mdx)[]&#62;

### Example

```ts
const fulfillmentSets = await fulfillmentModuleService.createFulfillmentSets(
  [
    {
      name: "Shipping",
      type: "default",
    },
    {
      name: "Pickup",
      type: "provider-controlled",
    },
  ]
)
```
